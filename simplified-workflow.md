# Setting Up Workflow in the New Project

After setting up your new project, you'll need to create a workflow to run it.

1. In your new Replit project, click on the "Tools" menu at the bottom
2. Select "Workflows"
3. Click "New workflow"
4. Name it "Start application"
5. Set the command to: `npm run dev`
6. Click "Create workflow"

This will create a workflow identical to the one in your current project, which you can use to run the simplified application.
